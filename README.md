# Netflix Visual Analytics (Python & R)

##  Project Overview
This project analyzes Netflix shows and movies using Python and R to extract insights through data cleaning, exploration, and visualization.

##  Applications Used
- Python (Pandas, Seaborn, Matplotlib)
- R (ggplot2)
- Jupyter Notebook / RStudio

##  Dataset
The dataset contains Netflix movies and TV shows including genres, ratings, release year, and country.

##  How to Run
### Python
1. Navigate to the `python` folder
2. Run `netflix_analysis.py`

### R
1. Open `netflix_visualization.R` in RStudio
2. Ensure the dataset path is correct
3. Run the script

##  Outputs
- Most Watched Genres
- Ratings Distribution (Python & R)
